메모장

파일에 원하는 메모를 저장하고 수정 및 조회가 가능하다.

python memo.py -v 를 입력하면 지금까지의 메모를 출력해준다.

python memo.py -a 를 입력하면 메모를 입력받아서 memo.txt에 적는다.

소스

import sys

import time

def usage():

print("""

Usage

=====

python %s -v : View memo

python %s -a : Add memo

""" % (sys.argv[0], sys.argv[0]))

if not sys.argv[1:] or sys.argv[1] not in ['-v', '-a']:

usage()

elif sys.argv[1] == '-v':

try: print(open("memo.txt").read())

except IOError: print("memo does not exist!")

elif sys.argv[1] == '-a':

word = input("Enter memo: ")

f = open("memo.txt", 'a')

f.write(time.ctime() + ': ' + word+'\n')

f.close()

print("Added")
