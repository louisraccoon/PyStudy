RSA 난수 생성

1.RSA 암호

RSA 암호는 공개키암호방식 중 한가지로 송신자가 공개되어 있는 공개키를 사용하여 암호화하고, 수신자가 자신만 가지고 있는 비밀키를 사용하여 복호화한다.

2.키 생성법

1.충분히 큰 서로 다른 소수 p,q를 선택한다.

2.pi(pq)=(p-1)(q-1) 를 구한다.

3.(p-1)과 (q-1)의 최소공배수 L을 구한다.

4.L보다 작으면서 서로소인 임의의 자연수 e를 선택한다.

5.ed=1(mod L) (max(p,q) < d < pi(pq)) 를 만족하는 d를 구한다.

3.코드

import random

#p값 입력

p = input('p = ')

p = int(p)

#q값 입력

q = input('q = ')

q = int(q)

#n = p*q

n = p*q

#pi = (p-1)*(q-1)

pi = (p-1)*(q-1)

#최대공약수

def gcd(a,b):

while(b!=0):

temp = a%b

a = b

b = temp

return abs(a)

#gcd(b,pi)==1 인 수를 구함

for i in range(2,pi-1):

if gcd(i,pi)==1:

b=i

break

#Zn*의 값들을 저장하기 위한 리스트

s = []

#s의 크기를 알기위한 변수

cnt_s = 0

#Zn*의 값들을 s리스트에 저장한다.

for j in range(2,n-1):

if gcd(j,n)==1:

s.append(j)

cnt_s = cnt_s+1

#Zn*의 값들중 랜덤으로 하나를 가져온다.

x0 = random.randrange(0,cnt_s)

#예제처럼 p=23, q =43을 입력했을때 11을 가져옴

#x0 = s[9]

#Output

print("Ouput")

#x의 값들을 저장하기 위한 리스트

x1 = x0**b%n

x = [x1]

#비트를 저장하기 위한 y 리스트

y1 = x[0]%2

y = [y1]

#주기 체크

cnt = 1

#난수 생성

for k in range(1,n):

x.append(x[k-1]**b%n)

y.append(x[k]%2)

cnt = cnt+1

if x[k] == x[0]:

break

print("X = ", x)

print("Y = ", y)

print("주기 = ", cnt)
