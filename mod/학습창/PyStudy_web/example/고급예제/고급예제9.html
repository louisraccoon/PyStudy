Display Calendar

달력 프로그램으로 보고싶은 년도와 달을 입력하면 원하는 달의 달력을 볼수있다.

소스

# import module

import calendar

# ask of month and year

yy = int(input("Enter year: "))

mm = int(input("Enter month: "))

# display the calendar

print(calendar.month(yy,mm))
